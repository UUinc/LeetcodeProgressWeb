# LeetcodeProgressWeb
 Flask web app for tracking group progress in LeetCode.com

### Build Command
```
pip install -r requirements.txt
```
### Start Command
```
python app.py
```

### Live App
You can test out the deployed app from this link:
https://leetcode-progress.onrender.com
